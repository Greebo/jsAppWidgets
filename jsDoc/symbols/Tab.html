Class Detail
Tab(source, style, caller)
Creates a tab object. The source div will be replayed by the generated tab 
var myTab = new Tab(   "myTab",           
                        { cssStyle         : 'simple',  
                          left             : '10px', 
                          top              : '10px',
                          width            : "250px",       
                          height           : "200px",   
                          selectedColor    : '#d48080', 
                          regAlign         : "left",   
                          regPos           : "top", 
                          regRows          : '1',      
                          regDistance      : '4px'     
                        }
                                );
Parameters:
{string} source
node or id of the source (html)
{hash} style
the style assignments for the tab
{integer} caller
optional parameter, if widget is called from an other widget
